# Project Overview

This project focuses on analyzing Netflix movie data using Python. The dataset was cleaned, preprocessed, and explored to uncover meaningful insights about movie genres, popularity, ratings, and release trends. A Power BI dashboard was also created to visualize the findings interactively.


# TOOL USED
Python
Pandas
NumPy
Seaborn
Matplotlib
Jupyter Notebook
Power BI

# DATA PREPROCESSING

The dataset was cleaned and prepared using Python.

Steps performed:

1. Removed missing values
2. Handled duplicate records
3. Converted data types
4. Cleaned release date information
5. Processed genre data
6. Prepared data for visualization and dashboard creation
